Class Caty::OptionConstructor
In: lib/caty/option_constructor.rb
Parent: Object

Constructs options using a DSL, e.g: 

  constructor.construct do
       desc('description')
       string  :option, 'default'

       desc('desc2')
       integer :nother_one
  end

Used by Caty when Caty::global_options() is called.
